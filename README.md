## HISTOIRE
### Au cours des années 1980:
**Bjarne Stroustrup**, pour sa thèse de doctorat, lui vient l'idée de créer un
nouveau language. Inspiré du language **Simula** arrive *C with classes* qui
plus tard *C++*.
